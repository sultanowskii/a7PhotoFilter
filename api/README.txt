Photo Filter API - API, которая принимает ваши фотографии и обрабатывает их. Особенность его заключается в том,
что он общается с помощью json, за счёт этого можно не ограничиваться лишь Telegram
(https://github.com/sultanowskii/a7PhotoFilter/tree/master/bot), а использовать его и на других платформах.

GitHub проекта: https://github.com/sultanowskii/a7PhotoFilter/tree/master/api

Этот API почти полностью соответствует REST-стандарту, и за счет этого обращение с ним
максимально просто и понятно - он поддерживает все базовые REST-функции.

Основные понятия: Модули User (пользователь), Image (изображение) и Room (комната).
Комната является хранилищем обработанных фото, в нее можно приглашать друзей и использовать ее вместе.
Пользователь может создать новую комнату или присоединится к уже существующей.
У всех членов конкретной комнаты есть доступ ко всем изображениям, которые находятся этой комнате.
